post /generarPresupuesto

Se Crea un presupuesto en la base de datos

  • Request
  • Response

Headers

  • RestKey: required (string)

    Example:

    clave1

Query Parameters

  • datos: required (object)
    • fechaPresupuesto: required (date-only)

      Example:

      2018-02-16
    • idCliente: required (integer)

      Example:

      2
    • referenciaProducto: required (string)

      Example:

      123ab
    • cantidadProducto: required (integer)

      Example:

      3

    Example:

    {
  "fechaPresupuesto": "2018-02-26",
  "idCliente": 1,
  "referenciaProducto": "123ab",
  "cantidadProducto": 2
}

Se ha creado un presupuesto

Body

Type: object

Example:

{

  "message" : "Correcto",

    }
}
